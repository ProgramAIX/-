# 八、表格
***表格第二行必须为空行，不然表格无效***
 
 title|t|e
|:----|----:|:---:|
|左对齐标题|右对齐标题|居中对齐标题|
- -、:之间的多余空格会被忽略，不影响布局。
- 默认标题栏居中对齐，内容居左对齐。
- -:表示内容和标题栏居右对齐，:-表示内容和标题栏居左对齐，:-:表示内容和标题栏居中对齐。
- 内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少有一个。
